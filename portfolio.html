DevTech Connect



| GitHub |

DevTech Connect is an application aimed at the web development community. It is a place for someone to start who is looking to learn a web language or technology. Resources abound when one hops on Google. We wanted to bring some orginization and focus to that.

This is the result of the first sprint. Please check back for updates.

Responsible for front-end development and design, which was done with React. Also created the content for all front-end focused learning tracks.

Borrowed



| GitHub | | Deployed |

Borrowed is a community web application, loosely modeled after LetGo or OfferUp, however, instead of users buying items from one another, they borrow items for a specified amount of time. The application operates at a very basic level at the moment. We do have plans for further development.

Please check back for updates.

Responsible for front-end development and design, which was done with Handlebars.js. Also set up the initial MVC and wrote a few routes for various button clicks.

DayTripper



| GitHub | | Deployed |

A web application that takes user location to find their ten-hour forecast. The user can then decide to go in or stay out and further choose from options. AJAX calls were written to search various APIs to help the user plan their day/evening.

Responsible for design as well as the reverse geocode, movie streaming, YouTube, and movies in theaters AJAX calls. Also intergrated all API responses to feed into the html dynamically.

Train App



 | GitHub | | Deployed |

A train schedule web application that takes in a train name, train destination, first deperature time, and time interval. The app then calculates the next train departure based on the current time. The JavaScript includes an object oriented, time processing script.

Node.js Liri



| GitHub |

Liri.js is a text-driven app mimicking Siri. Given commands in the terminal with Node.js, Liri responds. Commands are: my-tweets, spotify-this song {'song name here'}, movie-this {movie name here}, and do-what-it-says.
